# fast-ride

1. **Create the `.env` file**  
   At the root of the application, create a file named `.env`.

2. **Add the Google API key**  
   Inside the `.env` file, insert your Google API key in the following format:  
   ```env
   GOOGLE_API_KEY=<your-api-key>
   
3. **Start Application**
   ```bash
   docker compose up
   ```
